# 🛒 Blinkit Grocery Sales Dashboard | Power BI

## 📌 Overview

The **Blinkit Grocery Sales Dashboard** is a Power BI project built to analyze grocery sales data and provide insights into **business performance, product trends, and customer purchasing behavior**.

This project features a **single-page interactive dashboard** that delivers all key insights in one place.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Understand customer buying patterns
* Track key business KPIs

---

## 📊 Dashboard Features

This project includes **one interactive dashboard** with the following insights:

* 📈 Total Sales Revenue
* 🛒 Total Orders
* 💰 Average Order Value (AOV)
* 📊 Sales trends over time
* 🧺 Category-wise sales distribution
* ⭐ Top-selling products
* 👥 Customer purchase behavior

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **Power Query** (Data Cleaning & Transformation)
* **DAX** (Data Analysis Expressions)
* **Microsoft Excel Dataset**

---

## 🔄 Data Workflow

1. Imported data from Excel file
2. Cleaned and transformed data using Power Query
3. Built data model with relationships
4. Created DAX measures for KPIs
5. Designed a single-page interactive dashboard
